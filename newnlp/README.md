# Story2Audio Microservice

# Story to Audio Generator

This is a web application that converts a given story text into audio using Flask as the web framework and `gTTS` (Google Text-to-Speech) for audio generation. The user can input a story, select various parameters (accent, gender, emotion, and age group), and the application will generate an audio file of the story in the chosen voice and emotional tone.

## Features

- Convert text to speech with various customizable voices and emotions.
- Select from multiple accents (US, UK, Australia, Indian).
- Choose between different voices (Female, Male, Neutral).
- Choose emotions (Happy, Sad, Angry, Neutral).
- Choose an age group (Kid, Adult, Elder).
- Audio file is available for download once generated.

---

## Technologies Used

- **Flask**: A micro web framework for building web applications.
- **gTTS**: Google Text-to-Speech API used for generating speech from text.
- **HTML/CSS**: For the frontend design and user interface.
- **Python**: The backend logic for handling form submissions and audio generation.
- **UnitTest**: For testing the functionality of the app.
